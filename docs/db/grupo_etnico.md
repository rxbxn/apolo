# Tabla: grupo_etnico

Origen: consultado por MCP (mcp_supabase-apol_get_columns)

Columnas:

- id: bigint, NOT NULL — clave primaria (generated by default as identity)
- created_at: timestamp with time zone, NOT NULL — default now()
- nombre: text, NULL

Descripción:
Tabla simple para registrar los grupos étnicos. Se usa para el campo `grupo_etnico` en `militantes`.
